# X3L ToolBar
This program is a Multi-game modding toolkit, a various assortment of tools that you can use to mod some popular games.

## Features
* works with legit, unmodified steam version as well as with unpacked, not-so-legit versions

## Prerequisites
* .NET Framework 4.8
* administrative privileges (for patching)
* 64 bit OS
